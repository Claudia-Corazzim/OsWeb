Documentação da API

Clientes

GET /api/clientes

Retorna a lista de todos os clientes cadastrados.

Exemplo de resposta:

[
    {
        "id": 1,
        "nome": "João Silva",
        "telefone": "(11) 99999-9999",
        "email": "joao@email.com"
    }
]

Ordens de Serviço

GET /api/ordens

Retorna todas as ordens de serviço.

GET /api/ordens/<id>

Retorna uma ordem de serviço específica.

Exemplo de resposta:

{
    "id": 1,
    "cliente_nome": "João Silva",
    "data": "2024-01-20",
    "status": "Em andamento",
    "servicos": ["Troca de peça", "Manutenção"],
    "valores": [100.00, 50.00]
}

Estoque

GET /api/estoque

Retorna todos os itens do estoque.

Exemplo de resposta:

[
    {
        "id": 1,
        "descricao": "Peça XYZ",
        "quantidade": 10,
        "valor": 50.00
    }
]

Autenticação

Atualmente, a API é de acesso público para fins de demonstração. Em um ambiente de produção, seria necessário implementar autenticação.

Tratamento de Erros

A API retorna os seguintes códigos de status:

  • 200: Sucesso
  • 404: Recurso não encontrado
  • 500: Erro interno do servidor
